Tekla상에서 단위 중량 및 단면적을 구하는 방법

기본적으로 Tekla는 단면적을 기준으로 중량을 산출한다. 7850은 철의 비중이고, 10000으로 나누는 것은 중량 단위값을 (kg)으로 산출하기 위함이다.

작성 리스트 목록

  • Weight per unit length(kg)를 구하는 공식

    • Cross section area * 7850 / 10000 = Weight per unit length

    Cross section area * 0.785 = Weight per unit length

    Cross section area * (Length / 1000) * 0.785 = Weight per unit length

  • Cross section area(cm)를 구하는 공식(플레이트 두께,폭, 길이를 알고 있을 시)

    • thk * width / 100 = Cross section area

  • Cross section area(cm)를 구하는 공식(단위 중량을 알고 있을 시 구하는 공식)

    • thk * width / 100 = Cross section area

  • Tekla상에서 그레이팅 비중 계산(density(kg/m)를 구하는 공식)

    • thk * width * length * density = Weight per unit length

    thk * 1 * 1 * density = Weight per unit length

    density = Weight per unit length / ((thk * width * length) / 1000000000)

  • Tekla상에서 계단발판 비중 계산(density(kg/m)를 구하는 공식)

    • thk * width * length * density = Weight per unit length

    thk * 1 * 1 * density = Weight per unit length

    density = Weight per unit length / ((thk * width * length) / 1000000000)

  • Tekla상에서 특이한 형상(stud bolt)에 대한 비중 계산
    • 특이형상(즉 규격이 변경되지 않는 고정부재)에 대한 단위중량값이 존재하면, 테클라에서 기본으로 나오는 중량을 기준으로 비교 공식을 적용하여 비중을 산출한다.

    테클라 중량 : 7850 = 실제 중량 : 산출 비중

    (실제 중량 * 7850) / 테클라 중량 = 산출 비중
